Fregs API

The Fregs API is a thin public layer over the live collection. Use it to fetch minted Fregs, owner data and trait metadata from a clean HTTP surface, while the images and traits stay on-chain. The examples below are fetched live from this API so the code samples always match real responses.
Fregs start at token ID 0. If a token used to exist but was burned, the API returns 410 Gone.

Base URL https://api.fregs.io
Primary prefix /fregs
Response formats application/json, image/svg+xml
Live examples auto-load from the active API origin.
